# Deadline-Visualisierung
Eine Visualisierung, die zeigt, wie sich Abgabetermine, Klausuren und Deadlines über ein Semester verteilen. Nicht als Stressfaktor, sondern als Reflexion: Wie sieht ein typisches Semester eigentlich aus, wenn man alle Fristen auf einen Blick sieht? Wo häufen sich die Termine? Gibt es Muster?

Um Datenschutzbedenken zu vermeiden, arbeitet die Installation mit historischen oder anonymisierten Daten – zum Beispiel aggregierte Deadlines der letzten fünf Jahre, ohne Zuordnung zu einzelnen Personen oder Kursen. Oder eine manuelle Kuratierung besonders interessanter Semester.

Die Visualisierung selbst könnte als animierter Zeitstrahl laufen, als Heatmap, oder als abstraktere generative Darstellung, bei der Deadlines zu Partikeln, Wellen oder Verdichtungen werden.

Mögliche Technologien: Python für die Datenaufbereitung, D3.js oder p5.js für die Visualisierung. Als Datenquelle: exportierte Kalenderdaten aus dem Virtuellen Campus (in Absprache mit dem Rechenzentrum) oder manuell zusammengestellte historische Daten.
